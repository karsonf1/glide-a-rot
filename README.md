# Glide-A-Rot
First attempt at a roblox game, scripting assisted by Claude Code agent.
## Getting Started
To build the place from scratch, use:

```bash
rojo build -o "Hangglide.rbxlx"
```

Next, open `Hangglide.rbxlx` in Roblox Studio and start the Rojo server:

```bash
rojo serve
```

For more help, check out [the Rojo documentation](https://rojo.space/docs).
